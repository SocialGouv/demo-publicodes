    Présentation publi.codes

    Ce site web présente des exemples d'algorithmes encodés avec publi.codes.

    Publicodes est un langage déclaratif pour modéliser des domaines métiers complexes en les décomposant en règles élémentaires simples.

    Les algorithmes ainsi définis peuvent ensuite être exploités dans des applications web, mobiles, API... et être distribués via NPM ou GIT.
